# Schulnetz Plus Website

This folder contains the source code for the Schulnetz Plus landing page.

## How to publish on GitHub Pages

1.  **Push to GitHub**: Ensure this folder is pushed to your GitHub repository.
2.  **Rename Folder (Recommended)**: GitHub Pages works best if this folder is named `docs`.
    *   Rename `github website` to `docs`.
3.  **Configure Settings**:
    *   Go to your repository on GitHub.
    *   Click **Settings** > **Pages**.
    *   Under **Source**, select **Deploy from a branch**.
    *   Select your branch (e.g., `main`) and select the folder `/docs`.
    *   Click **Save**.

Your website will be live at `https://<your-username>.github.io/<repo-name>/`.

## Customizing

*   **index.html**: Edit the text, links, and structure.
*   **style.css**: Change colors, fonts, and layout.
